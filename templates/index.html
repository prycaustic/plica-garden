Welcome!

To get started, please follow these steps:

Create a folder named 'content' in the same directory as the server.

Inside the 'content' folder, you can organize your media using subfolders.

Folder structure

content
├── Notes
├── Moodboards
├── Documents

In this example, "Notes," "Images," and "Documents" will be displayed as separate pages on the left navigation tab. Each separate page will attempt to list its contents as appropriately as possible.

content
├── Notes
│   ├── todo-list.md
│   ├── homework.md
├── Moodboards
│   ├── Vacation
│   │   ├── image.png
│   │   ├── another-image.jpg
│   ├── Cats
│   │   ├── cat1.png
│   │   ├── funny-cat.gif

For example, inside of the 'Notes' directory you will see a list of notes grouped by their tags. If instead there is another folder with images (or videos) in it you will see a page with a list of moodboards.

Inside the 'Moodboards' directory you would see a list with the 'Vacation' moodboard and the 'Cats' moodboard. (MIXED NOTES AND MOODBOARDS EXPLANATION)

Enjoy!

Now, you are ready to use the app! Feel free to explore and manage your notes and images effortlessly.
